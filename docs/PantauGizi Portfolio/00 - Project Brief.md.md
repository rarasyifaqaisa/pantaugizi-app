# PantauGizi — Project Brief

## TL;DR
> Aplikasi nutrition tracker berbasis AI untuk makanan Indonesia.
> Foto makanan → AI deteksi → data kalori & nutrisi akurat.

## Problem
Aplikasi tracker populer tidak punya database makanan Indonesia.
28.7% penduduk Indonesia obesitas, tapi tidak ada tools lokal yang membantu.

## Solution
PantauGizi: foto → AI → nutrisi → insight harian & mingguan.

## Impact Target
- Pengguna tahu persis kalori nasi padang, gado-gado, soto ayam
- Tidak perlu input manual yang ribet
- Bisa dipakai langsung dari iPhone tanpa download app
