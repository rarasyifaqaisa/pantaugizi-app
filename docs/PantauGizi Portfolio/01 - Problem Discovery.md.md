# Problem Discovery

## Observasi Awal
Saya perhatikan teman-teman yang mau hidup sehat selalu frustrasi
karena tidak ada app yang punya data nasi padang atau gado-gado.

## Research
Saya test MyFitnessPal, Cronometer, FatSecret:
- Search "nasi goreng" → hasil tidak akurat / tidak ada
- Search "rendang" → tidak ditemukan
- Search "gado-gado" → versi western yang tidak relevan

**Kesimpulan:** <5% makanan Indonesia ada di database tracker populer.

## User Persona
**Nama:** Rizky, 22 tahun, mahasiswa
**Pain:** Mau diet tapi bingung kalori makanan Indonesia berapa
**Goal:** Tracking kalori tanpa ribet input manual

## Insight Utama
> "Orang Indonesia makan makanan Indonesia.
>  Kenapa tracker-nya pakai database orang barat?"

## Keputusan
Build nutrition tracker yang dimulai dari makanan lokal,
bukan menambahkan makanan lokal ke sistem yang sudah ada.
