# Reflection

## What Went Well
- Berhasil build full-stack app dari nol dalam waktu singkat
- PWA approach ternyata sangat viable untuk iOS experience
- Keyword matching system untuk AI adalah creative solution
  yang saya temukan sendiri lewat trial & error

## What I'd Do Differently
- Mulai dari Supabase SDK dari awal, bukan SQLAlchemy
- Setup CI/CD dari hari pertama agar deploy lebih smooth
- Test di iPhone lebih awal di proses development

## Biggest Learning
Constraint adalah creative catalyst. Tidak punya Mac, tidak punya
kartu kredit, tidak punya budget — semua "keterbatasan" ini justru
mendorong saya menemukan solusi yang lebih elegant dan accessible.

## What's Next
Kalau diterima di Apple Developer Academy, saya ingin:
1. Rebuild dengan SwiftUI untuk pengalaman iOS yang benar-benar native
2. Integrasi Apple HealthKit untuk sync dengan Health app
3. Gunakan Core ML untuk on-device food recognition (privacy-first)
4. Submit ke App Store
